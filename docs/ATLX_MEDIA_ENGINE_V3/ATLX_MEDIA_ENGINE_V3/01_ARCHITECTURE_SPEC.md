# 01 - ARCHITECTURE SPEC

## 1. Objective

ATLX Media Engine V3 is a shared service that receives, classifies, validates, processes, governs, and delivers media ready for consumption.

It is not a social network module. It is infrastructure.

## 2. Responsibilities

The engine does:

- Create media requests.
- Register media origin and rights.
- Register original uploaded file.
- Validate source, file, type, rights, and policy.
- Create processing jobs.
- Generate normalized media variants.
- Generate thumbnails and previews.
- Apply Antojados watermark only when policy allows it.
- Preserve external watermark when media originates from external platforms.
- Build ready payloads.
- Audit every meaningful action.
- Expose stable SP/API contracts.

The engine does not:

- Create posts.
- Create feeds.
- Compute ranking.
- Store comments or likes.
- Decide sponsorship plan.
- Decide city/metro ranking.
- Own Antojados business logic.
- Replace GT.
- Replace Antojados DB.

## 4. Ownership Boundary

Media Engine is the technical authority for media.

Antojados is the social/business authority for content usage.

The link is:

```text
antojados_post.media_id -> me.media_request.media_id
```

Antojados may copy final URLs for fast rendering, but must not mutate the technical meaning of media.

## 5. Official Origin Categories

- `official_antojados`: generated by AntojadosMX employees during assigned functions.
- `explorer_partner`: generated by external collaborators under license/contract.
- `created_in_antojados`: generated by normal users in Antojados.
- `uploaded_by_owner`: uploaded by owner/business/author.
- `licensed_content`: authorized content.
- `external_platform`: imported from TikTok/Instagram/Facebook/YouTube/etc.
- `demo_content`: temporary demo/reference content.
- `unknown`: not classified.

## 6. Key Policy Decisions

- Official Antojados employee content can be treated as company-owned if the employment/contract documentation supports it.
- Explorer partner content requires license/assignment contract.
- External platform content must preserve external watermark by default.
- Demo content must remain visibly labeled.
- Download must be disabled by default for external/demo content.
- Sharing should prefer links to Antojados, not direct file downloads.

## 7. Versioning

Contract version: `3.0.0`  
Payload version: `3`

Breaking changes require a new major version.
